# Развертывание бота на VPS — пошаговая инструкция

## Шаг 1: Купить сервер

Зайди на https://timeweb.cloud (или любой другой хостинг).

Параметры:
- ОС: **Ubuntu 22.04** или **24.04**
- Тариф: **минимальный** (1 ядро, 1 ГБ RAM) — от 199 руб/мес
- Регион: Москва (ближе к DaData API)

После создания получишь:
- IP-адрес (например `185.22.33.44`)
- Пароль root

---

## Шаг 2: Подключиться к серверу

На Mac открой Терминал и введи:

```
ssh root@185.22.33.44
```

(замени IP на свой)

Введи пароль — ты на сервере.

---

## Шаг 3: Загрузить файлы бота на сервер

**Не закрывай SSH.** Открой ВТОРОЕ окно Терминала на Mac и выполни:

```
scp -r /Users/vasya/клод\ ева/.claude/worktrees/youthful-zhukovsky/inn_checker_bot root@185.22.33.44:/root/
```

(замени IP на свой)

Это скопирует всю папку бота на сервер.

---

## Шаг 4: Запустить скрипт развертывания

Вернись в SSH-окно (первое) и выполни:

```
cd /root/inn_checker_bot
bash deploy.sh
```

Скрипт автоматически:
- Обновит систему
- Установит Python
- Создаст пользователя `botuser`
- Установит зависимости
- Настроит автозапуск
- Запустит бота

---

## Шаг 5: Проверить

Отправь ИНН боту в Telegram. Должен ответить.

---

## Полезные команды (выполнять на сервере)

```bash
# Проверить статус бота
systemctl status inn-checker-bot

# Посмотреть логи в реальном времени
journalctl -u inn-checker-bot -f

# Перезапустить бота
systemctl restart inn-checker-bot

# Остановить бота
systemctl stop inn-checker-bot

# Редактировать ключи API
nano /home/botuser/inn_checker_bot/.env
# (после изменения — systemctl restart inn-checker-bot)

# Посмотреть последние 50 строк лога
journalctl -u inn-checker-bot -n 50
```

---

## Что дает systemd

- Бот **автоматически запускается** при перезагрузке сервера
- Если бот **упал** — автоматически перезапустится через 5 секунд
- Все **логи** хранятся в системном журнале
- Бот работает **24/7** независимо от твоего компьютера

---

## Обновление бота

Если внес изменения в код на Mac:

```bash
# На Mac (заменить IP):
scp /Users/vasya/клод\ ева/.claude/worktrees/youthful-zhukovsky/inn_checker_bot/*.py root@185.22.33.44:/home/botuser/inn_checker_bot/

# На сервере:
ssh root@185.22.33.44
chown botuser:botuser /home/botuser/inn_checker_bot/*.py
systemctl restart inn-checker-bot
```
